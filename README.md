# 🖥️ Totem Integrado

Projeto em desenvolvimento de um **totem de autoatendimento integrado a uma balança comercial**, com foco em melhorar a experiência do cliente em ambientes de varejo e alimentação.
A proposta é permitir que o próprio cliente realize a pesagem de produtos, visualize valores em tempo real e finalize a compra de forma rápida e intuitiva.

> ⚠️ Status: Projeto em desenvolvimento — atualmente com foco em UX e Frontend.

## 🧭 User Flow

O fluxo do usuário foi projetado para proporcionar uma experiência de autoatendimento simples, rápida e intuitiva.

### Fluxo principal

1. **Tela inicial**
   - Apresentação do totem
   - Início da jornada do usuário

2. **Seleção de produtos**
   - Navegação pelos itens disponíveis
   - Escolha do produto para pesagem

3. **Integração com balança**
   - Usuário posiciona o produto
   - Sistema captura o peso automaticamente *(planejado)*

4. **Visualização de preço**
   - Cálculo automático baseado no peso
   - Feedback visual ao usuário

5. **Carrinho (atualmente em desenvolvimento)**
   - Listagem dos itens selecionados
   - Possibilidade de revisão

6. **Finalização da compra** *(planejado)*
   - Confirmação do pedido
   - Integração com pagamento

7. **Emissão de comprovante** *(futuro)*
   - Impressão ou exibição do recibo
   - Retorno à tela inicial
